)

Accept a pending connection.

Throws AcceptError on failure. The parameters are filled with the new socket's information. If sock is a valid socket then it is closed first. If the listener socket is set to non-block and there are no pending connections then the return value is false.

Parameters
addressReference to IpAddress instance where to put the IP address
portReference to a 16-bit unsigned integer where to put the port
sockReference to socket corresponding to the new connection
Returns
True on success, false if socket is non-blocking and there are no pending connections
