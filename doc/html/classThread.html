Detailed Description

A class used to manipulate threads.

You can create a thread in two ways: using the constructor, or by calling launch(). The function argument can return anything, but the value returned will be ignored and, therefore, lost.

Usage examples:

  • Non-member function
    float f()
    {
    // ...
    return 2.5f;
    }
    Thread t1(&f);
    // or
    Thread t2;
    t2.launch(&f);
  • Member function
    class C
    {
    public:
    void f(int x, double y, char* s)
    {
    // ...
    }
    };
    C obj;
    char s[20];
    Thread().launch(&C::f, obj, 25, 1.0, s).detach();
    // or
    Thread t2;
    t2.launch(&C::f, &obj, 0, 0.0, s + 3);
  • Functor
    class Functor
    {
    public:
    void operator() (int x)
    {
    // ...
    }
    };
    Thread t1(Functor(), 25);
    // or
    Functor obj;
    Thread t2;
    t2.launch(obj, 0);
  • Lambda
    Thread t1([] {std::cout << "t1\n";});
    // or
    Thread t2;
    t2.launch([](int x) {std::cout << "t2 " << x << '\n'}, 0);

◆ detach()

void Thread::detach ( )
inline

Detach the thread if it is joinable.

A detached thread will have its resources automatically released, but cannot be joined. This function will remove the object's ownership over the thread. If an error occurs then DetachError is thrown.
